## 프로젝트 소개
여행 기록과 여행 일정을 남기고 공유 하며 여행지마다 랭킹을 확인할 수 있는 웹페이지입니다.

## 개발기간
2024.1 ~ 2024.3



## 개발환경
React 18.2.0

IDE: vscode

## 인원
7명

 - 여행 기록 목록 표시

 - 글 수정
 - 페이지 모바일 환경 반응형 스타일링
